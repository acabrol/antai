# Antai User Stories

## 1. Purpose

This document lists user stories for workflows that are already represented in the current application.

Stories for deferred features should be added only when they are actively planned, and tracked first in [TASKS.md](/home/kalden/projects/antai/docs/TASKS.md).

## 2. Public and Account Stories

- As a guest, I want to understand what Antai does so that I can decide whether to use it.
- As a guest, I want to browse public projects and colonies so that I can explore public content before signing in.
- As a new user, I want to create an account so that I can keep my colony data private.
- As a returning user, I want to log in with my username or email so that I can continue my work quickly.
- As a user, I want to update my profile so that my account details stay correct.
- As a user, I want to choose my newsletter preference at signup so that communication stays optional.

## 3. Project and Colony Stories

- As an antkeeper, I want to create a project so that I can organize related colonies together.
- As an antkeeper, I want to create a colony so that I can start tracking it.
- As an antkeeper, I want to assign a species to a colony so that the colony is linked to care guidance.
- As an antkeeper, I want to edit colony details so that the record stays accurate over time.
- As an antkeeper, I want to browse all colonies so that I can quickly understand my setup.
- As an antkeeper, I want to open a colony detail view so that I can work on one colony in context.

## 4. Journal and Reminder Stories

- As an antkeeper, I want to add events to a colony so that I keep a reliable husbandry history.
- As an antkeeper, I want to edit events so that mistakes and new details can be corrected.
- As an antkeeper, I want to see recent activity on the dashboard so that I can catch up quickly.
- As an antkeeper, I want to create reminders so that I do not miss care tasks.
- As an antkeeper, I want to complete or snooze reminders so that task state reflects reality.
- As an antkeeper, I want urgent reminders surfaced clearly so that I know what needs attention now.

## 5. Species and Reference Stories

- As an antkeeper, I want to browse species entries so that I can learn care requirements.
- As an antkeeper, I want to open a species detail page so that I can read fuller guidance.
- As an antkeeper, I want species context visible while editing colonies so that data entry is easier.

## 6. Maps, Zones, and Photos

- As an antkeeper, I want to create a colony map so that I can represent my setup structure.
- As an antkeeper, I want to define zones so that parts of the setup can be described separately.
- As an antkeeper, I want to inspect zone details so that map structure remains understandable.
- As an antkeeper, I want to upload colony photos so that I can build a visual history.
- As an antkeeper, I want to review colony images in a gallery so that visual change is easier to track.

## 7. Sightings, News, and Public Pages

- As an antkeeper, I want to record a swarm sighting so that I can track field observations.
- As an antkeeper, I want to browse sightings so that I can see broader activity.
- As a user, I want to manage news feeds so that the content area stays useful.
- As a user, I want to browse public project and colony pages so that selected information can be shared.

## 8. AI Stories

- As an antkeeper, I want prepared prompts so that I can ask AI tools better questions with less setup.
- As an antkeeper, I want colony-aware AI context so that responses are grounded in real records.
- As a user, I want an AI assistant pane in the app so that AI workflows stay close to the data I am reviewing.
